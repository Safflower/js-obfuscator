# JavaScript Obfuscator

You can obfuscate JavaScript source code.

It uses only 9 different characters (`!+()[]{},`) to write.

And it supports Chrome, IE, and other browsers.

However, this works only with the http and https protocols.

Because this tool uses URL protocol as a gadget.
